File Layout:

1. Data Files - contains the set of files containing the data to be imported into the database. 

2. Data Import Instructions - contains instructions for how to use the data files to import data into the database

3. Diagrams - contains relational schema PDF

4. SQL
- INSERT_and_DELETE_samples: contains all of the sample INSERT and DELETE statements provided in our user manual.
- SQL_CREATE: contains all of the scripts needed to create your database schema on an empty database.
- SQL_QUERIES: contains all SQL queries from documentation.

5. bookStoreSchema.sql - contains the database

6. RelationalSchema.drawio - contains the relational schema drawio diagram

7.  CSE 3241 Final Project - contains the documentation for the project.
